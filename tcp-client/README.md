# tcp-client using mdns

# Software requirements
esp-idf v4.4/v5.0.   

# Installation
```
git clone https://github.com/nopnop2002/esp-idf-mdns
cd esp-idf-mdns/tcp-client/
idf.py menuconfig
idf.py flash
```
