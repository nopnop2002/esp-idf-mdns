# tcp-server using mdns

# Installation
```
git clone https://github.com/nopnop2002/esp-idf-mdns
cd esp-idf-mdns/tcp-server/
idf.py set-target {esp32/esp32s2/esp32s3/esp32c3}
idf.py menuconfig
idf.py flash
```
